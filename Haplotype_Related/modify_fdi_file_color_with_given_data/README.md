modify_fdi_file_color_with_given_data.py
========================================

Modify color, proportion of sector, and max node size of fdi file with given proportion data and fdi file.

Usage
-----

1. Open an terminal
2. Type: `python modify_fdi_file_color_with_given_data.py`
